\[ E[X]=Var[X]=\lambda. \]

Sin embargo, si la varianza muestral es mayor que la media muestral, hay indicios de sobredispersión y puede ser preferible una Binomial Negativa o un modelo con modificación en cero.

1.5 Ajuste Poisson Zero Modificada

El enunciado define

\[ X=(1-Z)(Y+1), \]

donde \(Z\sim Bernoulli(prob)\) y \(Y\sim Poisson(\lambda)\), independientes. Por tanto,

\[ P(X=0)=prob, \]

\[ E[X]=(1-prob)(\lambda+1), \]

\[ Var[X]=(1-prob)\lambda + prob(1-prob)(\lambda+1)^2. \]

La función de probabilidad es:

\[ P(X=0)=prob, \]

\[ P(X=x)=(1-prob)P(Y=x-1), \quad x=1,2,\ldots \]

# Función de masa de la Poisson Zero Modificada definida en el enunciado
# X = (1-Z)*(Y+1), Z Bernoulli(prob), Y Poisson(lambda)
dzmpois <- function(x, lambda, prob, log = FALSE) {
  dens <- ifelse(x == 0,
                 prob,
                 (1 - prob) * dpois(x - 1, lambda = lambda))
  if (log) log(dens) else dens
}

# Log-verosimilitud negativa para optimizar
negloglik_zm <- function(par, x) {
  lambda <- exp(par[1])
  prob <- plogis(par[2])
  -sum(dzmpois(x, lambda = lambda, prob = prob, log = TRUE))
}

# Valores iniciales razonables
prob_ini <- mean(nclaims == 0)
lambda_ini <- max(mean(nclaims[nclaims > 0]) - 1, 0.01)

fit_zm <- optim(par = c(log(lambda_ini), qlogis(prob_ini)),
                fn = negloglik_zm,
                x = nclaims,
                hessian = TRUE)

lambda_zm <- exp(fit_zm$par[1])
prob_zm <- plogis(fit_zm$par[2])

prob0_zm <- prob_zm
esp_zm <- (1 - prob_zm) * (lambda_zm + 1)
var_zm <- (1 - prob_zm) * lambda_zm + prob_zm * (1 - prob_zm) * (lambda_zm + 1)^2
loglik_zm <- -fit_zm$value
aic_zm <- -2 * loglik_zm + 2 * 2

c(lambda = lambda_zm,
  prob = prob_zm,
  prob0 = prob0_zm,
  esperanza = esp_zm,
  varianza = var_zm,
  AIC = aic_zm)
##       lambda         prob        prob0    esperanza     varianza          AIC

3.3 b) Ajuste Gamma

La distribución Gamma es adecuada para costes positivos y asimétricos. Ajustamos los parámetros por máxima verosimilitud.

fit_gamma <- fitdistr(X, densfun = "gamma")
fit_gamma
##       shape           rate    
##   5.2181106278   0.0051613659 
##  (0.3274367410) (0.0003335346)
shape_hat <- as.numeric(fit_gamma$estimate["shape"])
rate_hat <- as.numeric(fit_gamma$estimate["rate"])
scale_hat <- 1 / rate_hat

media_gamma <- shape_hat * scale_hat
var_gamma <- shape_hat * scale_hat^2

c(shape = shape_hat,
  rate = rate_hat,
  scale = scale_hat,
  media = media_gamma,
  varianza = var_gamma)
##        shape         rate        scale        media     varianza 
## 5.218111e+00 5.161366e-03 1.937472e+02 1.010994e+03 1.958772e+05

Representamos el histograma junto con la densidad ajustada.

Comprobamos el ajuste con el test de Kolmogorov-Smirnov.

ks_gamma <- ks.test(X, "pgamma", shape = shape_hat, rate = rate_hat)
ks_gamma

##

Si el p-valor es mayor que 0.05, no rechazamos que los costes procedan de una distribución Gamma. En ese caso, el ajuste Gamma puede considerarse razonable.

3.5 d) Densidad y distribución de X y de Yp

Para la variable por pago con franquicia ordinaria,

\[ F_{Y^P}(y)=\frac{F_X(y+d)-F_X(d)}{1-F_X(d)}, \quad y>0, \]

\[ f_{Y^P}(y)=\frac{f_X(y+d)}{1-F_X(d)}, \quad y>0. \]

fYp <- function(y) {
  dgamma(y + d, shape = shape_hat, rate = rate_hat) /
    (1 - pgamma(d, shape = shape_hat, rate = rate_hat))
}

FYp <- function(y) {
  (pgamma(y + d, shape = shape_hat, rate = rate_hat) -
     pgamma(d, shape = shape_hat, rate = rate_hat)) /
    (1 - pgamma(d, shape = shape_hat, rate = rate_hat))
}
xx <- seq(0, quantile(X, 0.995), length.out = 1000)

3.6 e) Probabilidad de que Yp esté entre 200 y 600

Queremos calcular

\[ P(200<Y^P<600)=P(600<X<1000\mid X>400). \]

prob_Yp_200_600 <- FYp(600) - FYp(200)
prob_Yp_200_600
## [1] 0.3943235

Usando únicamente pgamma, la probabilidad se calcula como:

prob_Yp_200_600_pgamma <-
  (pgamma(1000, shape = shape_hat, rate = rate_hat) -
     pgamma(600, shape = shape_hat, rate = rate_hat)) /
  (1 - pgamma(400, shape = shape_hat, rate = rate_hat))

3.7 f) Percentil 95 de Yp

El percentil 95 de \(Y^P\) es el valor \(q\) que cumple:

\[ F_{Y^P}(q)=0.95. \]

q95_Yp <- uniroot(function(y) FYp(y) - 0.95,
                  interval = c(0, max(X) * 10))$root
q95_Yp
## [1] 1446.573

El percentil 95 de la variable por pago es 1446.5735.

3.8 g) Límite máximo que afecta al 5 por ciento de los siniestros más caros

Si queremos que el límite afecte al 5 por ciento de los siniestros más caros teniendo en cuenta los costes originales, debemos calcular el percentil 95 de \(X\):

limite_original <- qgamma(0.95, shape = shape_hat, rate = rate_hat)
limite_indemnizacion <- limite_original - d

c(limite_original = limite_original,
  limite_indemnizacion = limite_indemnizacion,
  percentil95_Yp = q95_Yp)
##      limite_original limite_indemnizacion       percentil95_Yp 
##             1831.502             1431.502             1446.573

El límite sobre el coste original se establecería en 1831.5022 euros. Si se expresa como indemnización después de la franquicia, sería 1431.5022 euros.

Este valor no tiene por qué coincidir exactamente con el percentil 95 de \(Y^P\), porque \(Y^P\) está condicionado a que \(X>400\), mientras que el límite de este apartado se calcula sobre todos los costes originales.

3.9 h) Distribución de Yp aplicando también límite máximo

Definimos la indemnización con franquicia ordinaria y límite máximo sobre el coste original:

\[ Y_L=\min(X, L)-400 \mid X>400, \]

con \(L\) igual al percentil 95 de los costes originales.

La función de distribución coincide con la de \(Y^P\) hasta \(L-400\), y a partir de ese punto acumula una masa de probabilidad en el límite.

lim <- limite_original
lim_y <- lim - d

FYp_limitada <- function(y) {
  ifelse(y < lim_y,
         FYp(y),
         1)
}

y_grid <- seq(0, q95_Yp * 1.3, length.out = 1000)

plot(y_grid, FYp(y_grid),
     type = "l",
     lwd = 2,
     xlab = "y",
     ylab = "Función de distribución",
     main = "Distribución de Yp con y sin límite")
lines(y_grid, FYp_limitada(y_grid), lty = 2, lwd = 2)
abline(v = lim_y, lty = 3)
legend("bottomright",
       legend = c("Yp sin límite", "Yp con límite", "Límite"),
       lty = c(1, 2, 3),
       lwd = c(2, 2, 1))

prob_limitado_condicional <-
  (1 - pgamma(lim, shape = shape_hat, rate = rate_hat)) /
  (1 - pgamma(d, shape = shape_hat, rate = rate_hat))

prob_limitado_original <- 1 - pgamma(lim, shape = shape_hat, rate = rate_hat)

c(prob_limitado_original = prob_limitado_original,
  prob_limitado_condicionado_a_pago = prob_limitado_condicional)
##            prob_limitado_original prob_limitado_condicionado_a_pago 
##                        0.05000000                        0.05243653

La probabilidad de que un coste original supere el límite es 0.05, por construcción aproximadamente 0.05. Condicionando a que haya pago positivo, la probabilidad de que \(Y^P\) quede limitado por arriba es 0.05244.
